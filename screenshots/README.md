# Screenshots

This folder contains screenshots captured during scraper testing and debugging.

## Files

- `visaCal_failure.png` - Screenshot of the page when login or scxxxxxxxxxxls (for debugging)
- `visaCal_success.png` - Screenshot of successful login/dashboard

## Note

Screenshot files are generated during test runs and are not tracked by Git. Only this README is tracked to keep the folder structure in the repository.

To see current screenshots, run:
```bash
npm run test:connection
```

Screenshots will be saved to this directory if errors occur.
